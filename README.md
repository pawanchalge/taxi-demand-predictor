# 🚖 Taxi Demand Predictor (Traffic Demand Prediction)

## 📌 Overview

This project is a Machine Learning-based web application that predicts taxi demand (traffic demand) based on user input such as time or other features. It helps in understanding peak demand hours and can be useful for ride-sharing services and urban planning.

---

## 🎯 Objective

* Predict taxi demand using historical data
* Build a user-friendly web interface
* Deploy the model as a live web application

---

## ⚙️ Tech Stack

* **Python**
* **Flask** (Backend)
* **Scikit-learn** (Machine Learning)
* **Pandas & NumPy** (Data Processing)
* **HTML/CSS** (Frontend)

## ▶️ How to Run Locally

### Step 1: Clone the repository

```bash
git clone https://github.com/your-username/taxi-demand-predictor.git
cd taxi-demand-predictor
```

### Step 2: Install dependencies

```bash
pip install -r requirements.txt
```

### Step 3: Run the app

```bash
python app.py
```

### Step 4: Open in browser

```
http://127.0.0.1:5000
```
